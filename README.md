# BE:LON Collective V6

Komplette GitHub-/Vercel-Version mit reduzierter Bildwelt.

- Hero: dynamische Gruppenfahrt
- Manifesto: Pinguin-Illustration
- Constitution: ausschließlich Typografie und Prinzipien
- Collective: großes Umarmungsbild
- Ride It Forward: Einzelperson in der Landschaft
- Rides: Gruppe aus der Distanz
- Shop: Produktdarstellungen
- Newsletter: Bergpanorama

## Upload
1. ZIP entpacken.
2. GitHub → Add file → Upload files.
3. Den gesamten Inhalt hochladen.
4. Gleichnamige Dateien ersetzen.
5. Commit changes.

`package-lock.json` ist bewusst nicht enthalten.
